### Hearthstone_Project
유니티 3D 프로젝트로 제작한 하스스톤 입니다.

## 개발환경
* 유니티 2021.3.27f1
* VisualStudio 2022
* PhotonNetwork
* SourceTree

## 개발 기간
* 1인
* 2024.02.15 ~ 2024.05.22 (약 18주)

## 주요기능
+ 매칭시스템
  + 포톤 네트워크의 콜백함수와 유니티의 코루틴을 이용해서 매칭 시스템을 제작하였고 유저에게는 매칭 UI만 보이지만 실제로는 방을 찾아 들어가거나 방을 생성후 대기하는 기능이 실행됨
  + ![Matching](https://github.com/user-attachments/assets/ec770264-1310-476e-8295-486e543eadcd)
+ 카드관리
+ 플레이어 덱 생성및 저장
+ 게임 사이클
